##Phatformer

A 2D Platformer written in HTML5 for the Phatformer challenge, rules below.

##Phatformer Challenge

Make a game with the following features

* Platformer

* Gravity
* Collision Detection
* Projectile Launching
* Jet packs
* walking
* 2d
* bad guys
* scores
* health & death

bonus feature: multiplayer
